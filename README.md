# YOLOv8 目标检测 - 从零开始的新手完全指南

> 一名大一计算机新生的完整学习记录 | 从环境搭建到模型预测，涵盖所有踩坑经历与解决方案

本项目不仅是一份代码，更是一份**详尽的实验报告和教程**。它完整记录了我作为初学者，从零开始学习并使用YOLOv8实现小车目标检测的全过程。重点在于**环境配置、问题排查和原理理解**，旨在为后来者提供一份无坑的指南。

## 项目环境

-   **OS**: Windows 11
-   **Python**: 3.9
-   **Package Manager**: Anaconda
-   **深度学习框架**: PyTorch
-   **模型**: Ultralytics YOLOv8n
-   **显卡**: NVIDIA GeForce RTX 5060 (也包含CPU解决方案)

## 项目亮点

-   **真正从零开始**：面向零基础新手，无任何默认懂的前提。
-   **原理讲解**：不仅提供命令，更解释“为什么这么做”。
-   **全面排坑**：包含Anaconda安装冲突、CUDA版本兼容、网络下载失败、小样本过拟合等几乎所有常见问题的解决方案。
-   **命令字典**：几乎所有命令都配有参数解释，方便查阅。
